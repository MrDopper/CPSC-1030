What is HTML?

HTML is the standard markup language for Web pages.
With HTML you can create your own Website.
HTML is easy to learn - You will enjoy it!

HTML Elements

  • Each markup code represents an HTML element.
  • Each element has a purpose.
  • Most elements are coded as a pair of tags: an opening tag and a closing tag.
  • Tags are enclosed in angle brackets, "<" and ">" symbols

Document Type Definition (DTD)

  • doctype statement
  • identifies the version of HTML contained in your document.
  • placed at the top of a web page document
    • <! DOCTYPE html>

Heading in html

There are six types of heading: h1, h2, h3, h4, h5, and h6.
Each heading have different size. Bigger to smaller: h1 > h2 > h3 > h4 > h5 > h6
